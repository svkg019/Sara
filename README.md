## Installation
To install Sara, follow these steps:
- Clone the repository from GitHub using the following command:
~~~
git clone https://github.com/svkg019/Sara.git
~~~

- Then, open the `.env` file and update the API key.
- After successfully set the API key open the terminal in the current directory and run the following command:
~~~
python chatBotSara.py
~~~
